WinKnownPaths

This package allows you to reliably call known Windows paths identified by their UUID without having to make brittle calls to system environment variables.

Installation

pkg> add WinKnownPaths

Usage & Examples

Here's an example call to determine the system folder path and the font folder.

julia> using WinKnownPaths

julia> import WinKnownPaths: FOLDERID

julia> WinKnownPaths.path(FOLDERID.System)
"C:\\WINDOWS\\system32"

julia> WinKnownPaths.path(FOLDERID.Fonts)
"C:\\WINDOWS\\Fonts"

API

WinKnownPaths.path — Method
WinKnownPaths.path(folderid::UUID) -> AbstractString

Retrieve the full path of a known Windows path identified by the folder's id
specified in the module `FOLDERID`.

Examples

julia> import WinKnownPaths: FOLDERID

julia> WinKnownPaths.path(FOLDERID.System)
"C:\WINDOWS\system32"

julia> WinKnownPaths.path(FOLDERID.Fonts)
"C:\WINDOWS\Fonts"
